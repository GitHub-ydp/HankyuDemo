## Q1: PDF 合约解析是干什么的？

福山提供的资料中有一份 `LAX0751N25v93 (2).pdf`，这是 **ONE（Ocean Network Express）船司与阪急阪神美国子公司签署的 Service Contract（服务合约）**，共 **372 页**。

前几页是合约条款，后面几百页都是**费率表**，记录了从亚洲各港口到美国各港口的集装箱运价，例如：

```
ORIGIN: BANGKOK, THAILAND
Destination      20'    40'    40HC    45'
HONOLULU, HI    3840   4800   4800    6075
```

**PDF 合约解析的目标**：把这 372 页中散落的费率数据自动提取出来，入到数据库中。这样福山就不需要人工翻阅 PDF 去查找某条航线的费率。

## Q2: O/F、BAF/WRS、LSS 这些专业名词是什么意思？

### 柜型（Container Type）

| 缩写 | 全称 | 含义 |
|------|------|------|
| **20GP** | 20-foot General Purpose | **20英尺标准柜**，长约6米，最小的标准集装箱 |
| **40GP** | 40-foot General Purpose | **40英尺标准柜**，长约12米，是20GP的两倍长 |
| **40HQ/40HC** | 40-foot High Cube | **40英尺高柜**，长度同40GP，但高度多1英尺（约30cm），装更多货 |
| **45'** | 45-foot container | **45英尺特种柜**，比40还长，不常用，费率最贵 |

> 实际业务中 20GP 和 40GP/40HQ 是最常报价的柜型。40HQ 价格一般等于或略高于 40GP。

### 费率组成

| 缩写 | 全称 | 含义 |
|------|------|------|
| **O/F** | Ocean Freight | **海运费**，基本运费，船司把货从A港运到B港的核心费用 |
| **BAF** | Bunker Adjustment Factor | **燃油附加费**，因油价波动船司额外收取的费用。有时与 **WRS**（War Risk Surcharge，战争风险附加费）合并显示 |
| **LSS** | Low Sulphur Surcharge | **低硫附加费**，因IMO 2020规定船用燃油含硫量≤0.5%，低硫油更贵，船司将差价转嫁给客户 |

### 一条运价记录示例

以 KMTC 上海→釜山为例：

```
O/F:  20GP=$130  40GP=$260  40HQ=$260
BAF:  20GP=$220  40GP=$440
LSS:  含（已包含在O/F中）
```

- 运一个20英尺柜：海运费 $130 + 燃油附加费 $220 = **合计 $350**
- 运一个40英尺柜：$260 + $440 = **合计 $700**
- "含"表示低硫附加费已包含在海运费中，不另外收取

**客户最终付的运费 = O/F + BAF + LSS**，按不同柜型分别报价。

## Q3: 目前的需求中，除了海运还有别的运输方式吗？

需求文档中有一处提到"投标包内容包含：航线、**海运/空运**运费、有效期间、港杂费"。

但福山实际提供的全部数据（KMTC Excel、NVO FAK Excel、ONE PDF、长荣邮件、微信截图）**100%都是海运**。

**结论**：Demo 阶段只需覆盖海运。空运按 kg 计费，数据结构完全不同，如果未来有需求再扩展。

## Q4: NVO FAK 是什么？

- **NVO = NVOCC**（Non-Vessel Operating Common Carrier）= **无船承运人**。自己不拥有船，从船司批量买舱位后零售给货代或直客。
- **FAK** = Freight All Kinds = **不分货种的统一费率**（不管装什么货都是同一个价格）。

`NVO FAK 2026 (Mar 20 to Mar 31).xlsx` = NVO提供的统一运价表，有效期 2026/3/20~3/31。

### 文件结构（4个Sheet）

| Sheet | 全称 | 内容 |
|-------|------|------|
| TPE | Trans-Pacific Eastbound | 亚洲各港 → 美国西海岸（LA、Oakland等） |
| WPE | Westbound via Panama/East | 亚洲 → 美国东海岸（纽约、萨凡纳等） |
| Hawaii | 夏威夷专线 | 亚洲 → 檀香山等夏威夷港口 |
| Arbitrary | 内陆附加费 | 基本港转运到内陆城市的额外费用 |

### KMTC vs NVO FAK 对比

| | KMTC | NVO FAK |
|---|------|---------|
| 报价方 | 船司直接报价 | NVO（无船承运人）报价 |
| 航线范围 | 上海 → 韩国/东南亚/中东 | 亚洲多港 → 美国全境 |
| 起运港 | 固定上海 | 多个（新加坡、釜山、上海等14个港） |
| 有效期 | 每条单独标注 | 整个文件统一 |

阪急阪神两边都拿价，比价后选最优方案。
