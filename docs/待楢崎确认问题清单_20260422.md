# 待楢崎確認問題清單

- **版本**：v1.0
- **發布日期**：2026-04-22
- **彙總人**：Claude 監工
- **發起人**：張東旭（一次性向楢崎確認）
- **用途**：開發按清單"默認值"推進，**楢崎反饋後如有變更，按本清單回修**
- **紅線**：本清單裡所有問題，開發大師**不得自行拍板改動默認值**，必須先回來對標清單

---

## 使用方式

1. 開發過程中，凡遇到本清單裡的問題，按"默認值"列直接執行。
2. 楢崎有答覆時，在"楢崎回答"列填入，標記日期。
3. 答覆與默認值不一致時，創建回修任務（在 CLAUDE 任務列表裡新增 `T-FIX-Q*`）。
4. 建議**一次性問清**，分三批：P0（Demo 前必須答）/ P1（Demo 前建議答）/ P2（v2.0 討論）。
