# 启动说明

## Python 环境（重要）

### Mac（本地开发 — 当前使用）

项目根目录下有一个 Python 3.10 venv：

```
/Users/zhangdongxu/Desktop/project/阪急阪神/.venv
```

Python 解释器路径：`/Users/zhangdongxu/Desktop/project/阪急阪神/.venv/bin/python`
（指向 `/opt/homebrew/opt/python@3.10/bin/python3.10`）

首次装依赖：

```bash
cd /Users/zhangdongxu/Desktop/project/阪急阪神/backend
../.venv/bin/python -m pip install -r requirements.txt
```

### Windows（团队其他成员 / 备选）

使用 **Anaconda 的 py310 环境**，路径固定：

```
D:\Anaconda3\envs\py310
```

所有 Python 命令都要用这个环境的 `python.exe`，**不要用系统 Python**。

---

## 后端启动（FastAPI）

> ⚠️ 必须先 `cd backend` 再启动，否则 `.env` 不会被加载，会 fallback 到默认 PostgreSQL URL（连接失败）。

### Mac

```bash
cd /Users/zhangdongxu/Desktop/project/阪急阪神/backend
../.venv/bin/python -m uvicorn app.main:app --reload --port 8000
```

### Windows

```bash
cd backend
D:\Anaconda3\envs\py310\python.exe -m pip install -r requirements.txt
D:\Anaconda3\envs\py310\python.exe -m uvicorn app.main:app --reload --port 8000
```

启动后访问：
- API: http://localhost:8000
- Swagger 文档: http://localhost:8000/docs

---

## 前端启动（React + Vite）

```bash
cd frontend
npm install
npm run dev
```

启动后访问：http://localhost:5173

### Mac



```bash
cd backend && D:\Anaconda3\envs\py310\python.exe -m uvicorn app.main:app --reload --port 8000
```

**终端 2 — 前端：**

```bash

```
