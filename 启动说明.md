# 启动说明

## Python 环境（重要）

本项目使用 **Anaconda 的 py310 环境**，路径固定：

```
D:\Anaconda3\envs\py310
```

所有 Python 命令都要用这个环境的 `python.exe`，**不要用系统 Python**。

---

## 后端启动（FastAPI）

```bash
cd backend
D:\Anaconda3\envs\py310\python.exe -m pip install -r requirements.txt
D:\Anaconda3\envs\py310\python.exe -m uvicorn app.main:app --reload --port 8000
```

启动后访问：
- API: http://localhost:8000
- Swagger 文档: http://localhost:8000/docs

---

## 前端启动（React + Vite）

```bash
cd frontend
npm install
npm run dev
```

启动后访问：http://localhost:5173

---

## 一键全启（两个终端分别执行）

**终端 1 — 后端：**

```bash
cd backend && D:\Anaconda3\envs\py310\python.exe -m uvicorn app.main:app --reload --port 8000
```

**终端 2 — 前端：**

```bash
cd frontend && npm run dev
```

---

## 环境变量

后端依赖 `backend/.env`，已经配置好（包含 Qwen API Key、邮箱 IMAP 等）。
如需新建参考 `backend/.env.example`。
